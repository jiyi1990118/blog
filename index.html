hexo 下的标签云

发表于 2017-06-30 | 分类于 javascript

步骤一

Example - 新建一个页面，命名为 tags 
$ hexo new page "tags"

步骤二

Example - 编辑刚新建的页面，将页面的类型设置为 tags ，主题将自动为这个页面显示标签云
---
title: Tagcloud
date: 2014-12-22 12:39:04
type: "tags"
---
注意

如果有启用多说 或者 Disqus 评论，默认页面也会带有评论。需要关闭的话，请添加字段 comments 并将值设置为 false

Example - 配置
---
title: Tagcloud
date: 2017-6-30 12:39:04
type: "tags"
comments: false
---

步骤三

Example - 在菜单中添加链接。编辑 主题配置文件 ，添加 tags 到 menu 中
menu:
  home: /
  archives: /archives
  tags: /tags

任意获取他人qq头像网址 (支持动态)

发表于 2017-06-29 | 分类于 api

获取任意QQ好友动态头像 URL

Example - 请求的URL
http://q2.qlogo.cn/headimg_dl?bs=qq&dst_uin=QQ号码&src_uin=QQ号码&fid=QQ号码&spec=100&url_enc=0&referer=bu_interface&term_type=PC
More info:

把上面 “QQ号码” 都换成你们想要获取的QQ号，复制上面的址到浏览器里，即可以看到qq头像，如果是动态头像的话下载后加上.gif
